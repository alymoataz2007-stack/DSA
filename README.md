# Data Structures

This repository contains two C applications developed as part of my Data Structures coursework.

## Projects

### Spell Checker Using AVL Tree
- Dictionary implementation using an AVL Tree
- Fast word lookup and search
- Self-balancing tree operations

### Infix to Postfix Expression Evaluator
Supports:
- Single-digit numbers
- Multi-digit numbers
- Parentheses
- Floating-point numbers
- Negative numbers
- Exponentiation (Power operation)
